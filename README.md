# hnh

# Install

```
$ git clone https://github.com/WhiteCapella/hnh.git
$ cd hnh
$ source .venv/bin/activate
$ git checkout 0.2/mod_deep
```

# Usage

```
$ uvicorn src.hnh.app:app --reload
```
